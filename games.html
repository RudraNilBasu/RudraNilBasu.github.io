Fortior


Fortior is a puzzle platformer game made within 48 hours for the 33rd Ludum Dare (for the compo version), with the theme of You are the Monster. Fortior speaks out about the day-to-day monsters we face in our daily life. The project is open for contribution on Github

This Game went on to become the Runner's up for the NASSCOM Student Game of the Year Award (2015)
